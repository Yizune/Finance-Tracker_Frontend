**Finance Tracker Frontend**

This repository contains only the static frontend for the Finance Tracker application.

Follow these steps:

```sh
# 1. Clone the repository
git clone <YOUR_GIT_URL>
cd <repo-folder>

# 2. (Optional) Install dependencies if you want to build CSS from SCSS
npm install

# 3. (Optional) Build CSS from SCSS
npm run build:css
```

This project is built with:

- HTML 
- SCSS
- Vanilla JavaScript
- AnyChart

**Note:** The backend (API/server) is now a separate repository. This frontend does not include any backend/server code.
